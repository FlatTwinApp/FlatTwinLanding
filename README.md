# FlatTwin Landing Page

Static marketing website for FlatTwin.

## 🚀 Features
- **Waitlist:** Integrated with the backend `POST /leads` endpoint.
- **Responsive Design:** Optimized for mobile and desktop.
- **Style Guide Adherence:** Uses corporate colors (#4774D0) and Inter font.

## 🛠 Tech Stack
- HTML5 / CSS3
- Vanilla JavaScript
- Font: Inter (Google Fonts)

## 🔧 Backend Integration
The landing page points to the backend URL defined in the configuration to collect leads from the waitlist form.
